Qué es un sistema material

Programa representado: sistemas materiales, fase y componente.

Un sistema material es una porción de materia que seleccionamos para estudiar. Puede ser algo simple, como un vaso con agua, o más complejo, como agua con aceite y arena.

Para describir sistemas materiales usamos dos conceptos centrales: fase y componente. Una fase es cada parte visible con aspecto uniforme. Un componente es cada sustancia presente, se vea o no a simple vista.

Aprender esta diferencia ayuda a comunicar observaciones con precisión.

SVG dinámico: fase y componente

Ejemplos cotidianos

Agua con sal puede verse como una sola fase, aunque tenga más de un componente. Agua con aceite muestra dos fases porque las sustancias no se mezclan de forma uniforme.

Actividad breve

Observá tres sistemas de tu casa (por ejemplo jugo, leche con cacao, agua con aceite) y escribí cuántas fases visibles tiene cada uno.
